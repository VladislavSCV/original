# Система бронирования банкетных залов

## Ссылка на репозиторий - https://github.com/VladislavSCV/original

## Описание проекта
Веб-приложение для автоматизации процесса бронирования помещений под проведение банкетных и праздничных мероприятий.

## Архитектура базы данных

### Схема таблиц
* **`users`** — Учетные записи пользователей (авторизация, роли, права администратора).
* **`bookings`** — Заявки на бронирование помещений.
* **`reviews`** — Отзывы и обратная связь пользователей к записям.

### Структура таблицы бронирования (`bookings`)


| Поле | Тип данных | Описание |
| :--- | :--- | :--- |
| `id` | `bigint` (system) | Уникальный идентификатор записи |
| `user_id` | `bigint` (system) | Идентификатор создателя заявки |
| `created_at` | `timestamp` (system) | Дата и время создания брони |
| `room_type` | `enum` | Категория/тип выбранного помещения |
| `start_date` | `date` | Планируемая дата проведения банкета |
| `payment_method` | `enum` | Предпочтительный способ оплаты |
| `status` | `varchar` | Текущий статус обработки заявки |

## 🚀 Инструкция по запуску

Для локального запуска проекта выполните следующие команды в терминале:

### Бэкенд (Go)
```bash
cd backend
go mod tidy
go run main.go
```

### Фронтенд (Node.js)
```bash
cd frontend
npm install
npm run dev
```
